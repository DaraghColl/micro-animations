# Micro Animations

A growing list of micro animations, based on [Lucide icons](https://lucide.dev/icons/) and utilising [Motion](https://motion.dev/docs/quick-start).

## Features

- Hover to play animation
- Click to copy code
- Built with TypeScript and Vite
- Modern, responsive UI

## Getting Started

### Prerequisites

- Node.js (v16 or higher recommended)
- npm or yarn

### Installation

```bash
npm install
```

### Development

To start the development server:

```bash
npm run dev
```

### Build

To build for production:

```bash
npm run build
```

## Project Structure

- `src/` — Source code (TypeScript, React components, CSS)
- `public/` — Static assets
- `index.html` — Main HTML file
